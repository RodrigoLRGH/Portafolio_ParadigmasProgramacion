Práctica 4: Paradigma logico

Introducción

Prolog o PROgramming en LOG, es un lenguaje de programacion logico y declarativo.

Funciona mas como una base de conocimientos donde dices que cosas son verdaderas y como se relacionan. Esto permite responder preguntas preguntas haciendo una busqueda (consultando) en una base de hechos y reglas para deducir una respuesta (si es que se puede).

El leguaje Prolog tiene tres elementos diferentes:

Hechos: El hecho es un predicado que es verdadero.

Reglas: Las reglas son extinciones de hechos que contienen clausulas condicionals. Para satisfacer una regla, estas condiciones deben cumplirse

Preguntas: Para ejecutar un programa prolog preguntas, estas preguntas pueden responderse mediante los hechos y las reglas dadas.
